# Configuration

تمثل فئات التكوين الطريقة التي يتم من خلالها تحديد كيفية تنفيذ مهمة ما. هناك مهمتان مدعومتان في حزمة ONNX Runtime:

1. التحسين: يقوم به [`~onnxruntime.ORTOptimizer`]، ويمكن ضبط هذه المهمة باستخدام [`~onnxruntime.configuration.OptimizationConfig`].

2. التكميم: يقوم به [`~onnxruntime.ORTQuantizer`]، ويمكن تحديد التكميم باستخدام [`~onnxruntime.configuration.QuantizationConfig`]. وتتطلب بعض الحالات (مثل التكميم الثابت بعد التدريب) خطوة معايرة، والتي يمكن تحديدها باستخدام [`~onnxruntime.configuration.CalibrationConfig`].
