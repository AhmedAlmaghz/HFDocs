# نظرة عامة

🤗 Optimum يتولى تصدير نماذج TensorFlow إلى TFLite في وحدة `exporters.tflite`. بالإضافة إلى ذلك، ستتم أيضًا دعم النماذج المستضافة على Hugging Face Hub بوزن PyTorch ولكن مع وجود تنفيذ TensorFlow في التصدير بفضل تحويل Transformers' [TFPreTrainedModel.from_pretrained()](https://huggingface.co/docs/transformers/main/en/main_classes/model#transformers.TFPreTrainedModel.from_pretrained) التلقائي إلى TensorFlow.

يوفر دعم تصدير TFLite الفئات والوظائف وواجهة سطر الأوامر لتصدير النموذج بسهولة.

التصميمات المدعومة:

- Albert
- BERT
- Camembert
- ConvBert
- Deberta
- Deberta V2
- DistilBert
- Electra
- Flaubert
- MobileBert
- MPNet
- ResNet
- Roberta
- RoFormer
- XLM
- XLMRoberta
