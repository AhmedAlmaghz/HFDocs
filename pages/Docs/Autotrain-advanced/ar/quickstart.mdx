# بداية سريعة

هذه البداية السريعة هي للتثبيت والاستخدام المحليين. إذا كنت تريد استخدام AutoTrain على Hugging Face Spaces، يرجى الرجوع إلى قسم "AutoTrain على Hugging Face Spaces".

يمكنك تثبيت AutoTrain Advanced باستخدام pip:

```bash
$ pip install autotrain-advanced
```

من المستحسن تثبيت autotrain-advanced في بيئة افتراضية لتجنب أي تعارض مع الحزم الأخرى.

ملاحظة: لا يقوم AutoTrain بتثبيت pytorch أو torchaudio أو torchvision أو أي تبعيات كبيرة أخرى. ستحتاج إلى تثبيتها بشكل منفصل.

```bash
$ conda create -n autotrain python=3.10
$ conda activate autotrain
$ pip install autotrain-advanced
$ conda install pytorch torchvision torchaudio pytorch-cuda=12.1 -c pytorch -c nvidia
$ conda install -c "nvidia/label/cuda-12.1.0" cuda-nvcc
$ conda install xformers -c xformers
$ python -m nltk.downloader punkt
$ pip install flash-attn --no-build-isolation # إذا كنت تريد استخدام flash-attn
$ pip install deepspeed # إذا كنت تريد استخدام deepspeed
```

# تشغيل واجهة مستخدم AutoTrain

لتشغيل تطبيق autotrain محليًا، يمكنك استخدام الأمر التالي:

```bash
$ export HF_TOKEN=your_hugging_face_write_token
$ autotrain app --host 127.0.0.1 --port 8000
```

سيبدأ هذا تشغيل التطبيق على `http://127.0.0.1:8000`.

# استخدام واجهة سطر أوامر AutoTrain

من الممكن أيضًا استخدام واجهة سطر الأوامر:

```bash
$ export HF_TOKEN=your_hugging_face_write_token
$ autotrain --help
```

سيتم عرض أوامر واجهة سطر الأوامر التي يمكن استخدامها:

```bash
usage: autotrain <command> [<args>]

positional arguments:
{
app,
llm,
setup,
dreambooth,
api,
text-classification,
text-regression,
image-classification,
tabular,
spacerunner,
seq2seq,
token-classification
}

commands

options:
-h, --help            show this help message and exit
--version, -v         Display AutoTrain version
--config CONFIG       Optional configuration file

For more information about a command, run: `autotrain <command> --help`
```

من المستحسن استخدام الأمر `autotrain --config CONFIG_FILE` عند استخدام واجهة سطر الأوامر.

أوامر autotrain التي قد يهتم بها المستخدمون النهائيون هي:

- `app`: بدء واجهة مستخدم AutoTrain
- `llm`: تدريب نموذج لغة
- `dreambooth`: تدريب نموذج باستخدام DreamBooth
- `text-classification`: تدريب نموذج تصنيف النص
- `text-regression`: تدريب نموذج تراجع نصي
- `image-classification`: تدريب نموذج تصنيف الصور
- `tabular`: تدريب نموذج جدولي
- `spacerunner`: تدريب أي نموذج مخصص باستخدام SpaceRunner
- `seq2seq`: تدريب نموذج تسلسل إلى تسلسل
- `token-classification`: تدريب نموذج تصنيف الرموز

ملاحظة: لا يلزم استخدام الأوامر المذكورة أعلاه إذا كنت تستخدم الأمر المفضل `autotrain --config CONFIG_FILE` لتدريب النماذج.
