# تصنيف الرموز 

يعد تصنيف الرموز مهمة لتصنيف كل رمز في تسلسل. يمكن استخدام هذا الأمر لتعيين الكيانات المسماة (NER) ووضع علامات على أجزاء الكلام، وأكثر من ذلك. قم بإعداد بياناتك بالشكل المناسب، ثم بنقرات قليلة، سيتم إعداد نموذجك المتقدم للاستخدام في الإنتاج.

## تنسيق البيانات

يجب أن تكون البيانات بتنسيق CSV التالي:

```csv
tokens,tags
"['I', 'love', 'Paris']", "['O', 'O', 'B-LOC']"
"['I', 'live', 'in', 'New', 'York']", "['O', 'O', 'O', 'B-LOC', 'I-LOC']"
.
.
.
```

أو يمكنك أيضًا استخدام تنسيق JSONL:

```json
{"tokens": ["I", "love", "Paris"], "tags": ["O", "O", "B-LOC"]}
{"tokens": ["I", "live", "in", "New", "York"], "tags": ["O", "O", "O", "B-LOC", "I-LOC"]}
.
.
.
```

كما ترون، لدينا عمودان في ملف CSV. أحد الأعمدة هو الرموز والآخر هو العلامات. كلا العمودين عبارة عن قوائم مسلسلة! يحتوي عمود الرموز على رموز الجملة ويحتوي عمود العلامات على العلامات لكل رمز.

إذا كان ملف CSV الخاص بك كبيرًا، فيمكنك تقسيمه إلى عدة ملفات CSV وتحميلها بشكل منفصل. يرجى التأكد من أن أسماء الأعمدة هي نفسها في جميع ملفات CSV.

إحدى طرق تقسيم ملف CSV باستخدام الباندا هي كما يلي:

```python
import pandas as pd

# Set the chunk size
chunk_size = 1000
i = 1

# Open the CSV file and read it in chunks
for chunk in pd.read_csv('example.csv', chunksize=chunk_size):
# Save each chunk to a new file
chunk.to_csv(f'chunk_{i}.csv', index=False)
i += 1
```

## الأعمدة

يجب أن يحتوي ملف CSV/JSONL على عمودين: `tokens` و `tags`.
