# التصنيف/الانحدار الجدولي

باستخدام AutoTrain، يمكنك تدريب نموذج لتصنيف أو انحدار البيانات الجدولية بسهولة. كل ما عليك فعله هو اختيار نموذج من قائمة النماذج وتحميل مجموعة البيانات الخاصة بك. يتم ضبط المعلمات تلقائيًا.

## النماذج

تتوفر النماذج التالية للتصنيف/الانحدار الجدولي:

- xgboost
- random_forest
- ridge
- logistic_regression
- svm
- extra_trees
- gradient_boosting
- adaboost
- decision_tree
- knn
